## 启动项目 [​](#启动项目)

本项目使用 pnpm 工具安装依赖，推荐使用 pnpm

```
npm install -g pnpm
# 或者
yarn global add pnpm
```
### 安装依赖 [​](#安装依赖)

```
pnpm install
```
### 运行 [​](#运行)

```
pnpm dev
```

项目启动后会自动打开浏览器运行
